# js-fizz-buzz

`js-fizz-buzz` is a Node.js JavaScript implementation of fizz buzz.

## Setup

Install dependencies:

```
yarn install
```

Build code:

```
yarn build
```

## Usage

Start for dev (from `src` with `babel-node` and `nodemon`):

```
yarn start:dev
```

Start for prod (from `dist` with `node`):

```
yarn start:prod
```

Run tests:

```
yarn test
```

Lint code:

```
yarn lint
```

Format code:

```
yarn format
```
